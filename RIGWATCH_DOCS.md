# RigWatch Documentation

> **Status:** scaffold — full docs ship in a later phase.

## 1. What is RigWatch?

RigWatch is a real-time monitoring and operations dashboard for cryptocurrency
mining rigs. Operators connect to rigs by ID, watch live telemetry (hashrate,
temperature, power, fan speed, share rejection), receive alerts, and run health
diagnostics.

## 2. How the app is organised

The app has three layouts:

- **Standard mode** — full operator dashboard with charts, parameter grid, and admin tools.
- **Simple mode** — minimal read-only view for owners / investors.
- **Operator mode** — specialised view for on-call engineers, with AI-assisted diagnostics.

Authentication is wallet-based (demo): connect any wallet to claim ownership of
your rigs and access protected actions.
